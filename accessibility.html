Accessibility Statement

Last updated: January 2026

Commitment to Accessibility

Plowshare is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Accessibility Standards

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards. These guidelines explain how to make web content more accessible to people with disabilities.

Accessibility Features

Our website includes the following accessibility features:

  • Alternative text descriptions for images
  • Keyboard navigation support throughout the site
  • Clear heading structure for screen readers
  • Sufficient color contrast ratios for readability
  • Responsive design that works on all devices
  • Form labels and error messages clearly identified
  • Skip navigation links for keyboard users
  • Accessible PDF documents

Third-Party Services

Some content on our website may be provided by third-party services. While we work with our partners to ensure accessibility, we cannot guarantee that all third-party content fully meets WCAG 2.1 Level AA standards.

Known Accessibility Issues

We are aware of the following accessibility challenges and are actively working to resolve them:

  • Some video content may not include captions
  • Certain interactive elements may require keyboard accessibility improvements

Assistive Technology Support

Our website has been tested with the following assistive technologies:

  • Screen readers (NVDA, JAWS, VoiceOver)
  • Keyboard navigation
  • Browser zoom functionality
  • High contrast modes

Browser and Device Support

Our website is designed to work with the following browsers:

  • Chrome (latest version)
  • Firefox (latest version)
  • Safari (latest version)
  • Edge (latest version)

How to Use Accessibility Features

Keyboard Navigation

You can navigate through the website using only your keyboard:

  • Tab: Move to the next link or button
  • Shift + Tab: Move to the previous link or button
  • Enter: Activate a link or button
  • Escape: Close menus or dialogs

Screen Reader Tips

When using a screen reader, headings are marked with heading levels (H1, H2, etc.) to help you navigate the page structure. Use your screen reader's heading navigation commands to jump between sections.

Adjusting Text Size

You can adjust the text size in your browser:

  • Windows/Linux: Ctrl + Plus (+) to increase, Ctrl + Minus (-) to decrease
  • Mac: Command + Plus (+) to increase, Command + Minus (-) to decrease

Feedback and Accessibility Issues

If you encounter any accessibility barriers on our website, please let us know. We value your feedback and will work to resolve any issues:

  • By email: accessibility@plowshare.com
  • By phone: (000) 000-0000
  • By visiting our contact page

Ongoing Improvements

Accessibility is an ongoing process. We are committed to regularly reviewing and improving the accessibility of our website. If you have suggestions for how we can improve, please don't hesitate to contact us.
